## **Tracker**

Tracker — приложение для формирования полезных привычек и отслеживания регулярных активностей. Пользователь создаёт трекеры, назначает им расписание и отмечает выполнение каждый день. По итогам ведения трекеров доступна статистика.


## **Описание приложения**

- Приложение-трекер привычек, в котором пользователь создаёт трекеры — регулярные привычки — и отслеживает их выполнение по дням.
- Трекеры можно группировать по категориям, задавать им расписание по дням недели и отмечать как выполненные для выбранной даты.
- На главном экране отображается список трекеров, актуальных для выбранной даты, с возможностью поиска и фильтрации.
- После накопления данных пользователь может посмотреть личную статистику: количество выполненных трекеров.

## **Функциональные требования**

- При первом запуске приложения показывается экран онбординга;
- На главном экране отображаются трекеры, запланированные на выбранную дату; дата выбирается через датапикер;
- Трекеры разделены на категории; каждый трекер относится к одной категории;
- Пользователь может создать трекер-привычку с заданием расписания по дням недели;
- Для каждого трекера можно задать название, цвет, эмодзи и категорию;
- Трекер можно отметить как выполненный для выбранной даты; повторное нажатие снимает отметку;
- Пользователь может редактировать и удалять трекеры через контекстное меню (долгое нажатие);
- Доступна фильтрация трекеров: Все, Сегодня, Завершённые, Незавершённые;
- Реализован поиск трекеров по названию;
- На экране статистики отображаются: количество завершённых трекеров;
- Взаимодействие пользователя с приложением отслеживается с помощью аналитики.

## **Технические требования**

- Приложение поддерживает устройства iPhone с iOS 17; предусмотрен только портретный режим;
- Язык реализации — Swift; UI-фреймворк — UIKit;
- Архитектура — MVC;
- Хранение данных — Core Data;
- Аналитика — AppMetrica (YandexMetrica);
- Приложение локализовано и поддерживает несколько языков.
